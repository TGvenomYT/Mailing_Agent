# 📧 MailingAgent – Your Smart Email Assistant

![MailingAgent Banner](https://img.shields.io/badge/MailingAgent-📬-blueviolet)  

**MailingAgent** is a **Python-powered email automation and management tool**.  
Send emails, summarize inbox content, detect spam, and even generate AI-powered email bodies — all in one neat package.  

---

## 🌟 Features

- ✉️ **Send Emails** – With or without attachments  
- 🤖 **AI-generated Email Body** – Generate concise, context-aware content  
- 📬 **Inbox Summarization** – Summarize unread Gmail messages  
- 🚫 **Spam Detection** – Identify unwanted emails using ML  
- 🌐 **Optional Web Integration** – Scrape content and fetch news  

---

## ⚡ Quick Start

### 1️⃣ Install Dependencies
```bash
git clone https://github.com/YourUsername/MailingAgent.git
cd MailingAgent
pip install -r requirements.txt
```
```
HEAD TO .ENV FILE AND EDIT THE FIELDS

export SENDER_EMAIL="your_email@gmail.com"
export EMAIL_PASSWORD="your_app_password"
export SMTP_SERVER="smtp.gmail.com"
export SMTP_PORT="465"
```

##💻 Tech Stack
```
| Technology            | Purpose                     |
| --------------------- | --------------------------- |
| Python 3.12+          | Core programming language   |
| IMAPClient            | Fetch and read emails       |
| smtplib + SSL         | Secure email sending        |
| pandas + scikit-learn | Spam detection using ML     |
| LangChain + Ollama    | AI-powered email generation |

```
